** CLASE DE GIT **
- set-upstream: Comuncación directa sincrónizada con el repositorio remoto. Cuando se le haga push, haga todo lo necesario para que se haga "realidad" el push
- Mezcla - merge:  
el "git push -u ..." la parte de "-u" significa que se hace una canal d ecomunicación por defecto para que cuando se haga push vaya automaticamente al repositorio que se quiera. Solo se hace una vez
Normalmente se hace git push origin <name>

Consejo para imágenes:
- Se colocan con displaay: block; width = 100%;

margin-block: Agregar margen arriba y abajo
